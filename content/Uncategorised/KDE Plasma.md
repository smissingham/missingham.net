# Shortcuts

## General

| Default Shortcut     | Custom Binding | Description                        |
| -------------------- | -------------- | ---------------------------------- |
| **Alt + Space**      |                | Run Command Interface              |
| **Ctrl + F10**       | **Meta + Tab** | Present Windows                    |
| **Ctrl + F9**        |                | Present Windows on Current Desktop |
| **Ctrl + Alt + Del** |                | Logoff / Reboot Command Interface  |
|                      |                |                                    |
